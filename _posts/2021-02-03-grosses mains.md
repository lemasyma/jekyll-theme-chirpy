# Concept

* Object displacement
* Mutliple scales

# Interactable

# Keyboard/mouse player
* Input System Manager
* Cimemachine Camera
    * Handles Mouse input
* Define movement direction according to the camera vectors
* Handles rotation and animations
* Add jump and gravity

# Network handling

* Verifier que perso network bien instancie
